# Mini-Bank
# 🏦 Mini Bank Management System

A simple and efficient **Bank Management System** developed using **Java**. This project demonstrates core banking operations such as account creation, deposits, withdrawals, balance checking, and transaction management through a console-based application.

---

# 📌 Features

- ✅ Create New Bank Account
- 💰 Deposit Money
- 💸 Withdraw Money
- 📊 Check Account Balance
- 🔍 View Account Details
- 🔐 Basic Validation & Error Handling
- 🖥️ Console-Based User Interface

---

# 🛠️ Technologies Used

- **Java**
- **OOP Concepts**
- **Java Scanner Class**
- **Exception Handling**


---

# 🧠 Concepts Covered

- Classes & Objects

- Exception Handling
- User Input Handling
- Data Management
